网络配置

1、语音平台服务器IP地址 （必填项*）

语音接入服务的网络地址

FreeSwitch服务器的网络IPv4地址，如果是其他厂商的设备，请填写软电话的接入地址

2、语音平台服务器地址 （必填项*）

语音接入服务的网络地址

FreeSwitch服务器的网络地址，如果是其他厂商的设备，请填写软电话的接入地址

3、语音平台服务端口

语音接入服务的网络端口

FreeSwitch服务器的网络端口，如果是其他厂商的设备，根据需要填写

4、语音平台服务访问密码

语音接入服务的网络端口

FreeSwitch服务器的网络访问密码，如未修改，无需填写

3、启用软电话条的WebRTC功能

启用此功能后，坐席软电话条将通过WebRTC处理语音功能

包括拨打电话和接听电话

checked="checked">
4、语音设备类型

语音设备的类型，按照厂商选择

不同厂商设备的配置参数和链接方式可能会不同

3、坐席就绪时分配最大访客数量

为每个人工坐席分配访客的最大数量

为避免坐席就绪的时候突然涌入大量的咨询客户，设置此参数

5、客户超时提醒

客户与坐席对话的超时提醒功能

开启后，用户在指定时间内没有响应时，系统会自动结束对话

checked="checked">

超时时长（秒）

超时提示消息

6、客户超时提醒后再次超时

客户与坐席对话超时后再次超时后出策略

开启后，用户在指定时间内没有响应时，系统会在2分钟后自动结束对话，

checked="checked">

提醒后再次超时时长（秒）

提醒后再次超时后断开的提示消息

7、坐席回复超时

坐席回复客户消息的超时自动回复功能

开启后，坐席在指定时间内没有回复消息时，系统会自动回复用户消息

checked="checked">

超时时长（秒）

超时提示消息
